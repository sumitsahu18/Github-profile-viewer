# GitHub Profile Viewer

A simple and responsive **GitHub Profile Viewer** built using **HTML, CSS, and JavaScript**.  
It fetches real-time user data using the **GitHub API** and displays complete profile details.

# Features 
1.  Search any GitHub username   
2.  Display user avatar & full name   
3.  Show user bio   
4.  Total public repositories   
5.  Followers count   
6.  Following count   
7.  List of repositories with links   
8.  Fast and clean UI   

# Technologies Used 
1. HTML5   
2. CSS3   
*JavaScript (Fetch API)*
-*GitHub REST API*

# How to Use 
1. Enter a valid GitHub username in the input box  
2. Click the search button  
3. The user profile info will appear instantly  

# API Used 
GitHub API: `https://api.github.com/users/${username}`
